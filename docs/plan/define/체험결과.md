# 체험 조사 결과

## 조사 개요

- **조사 목적**: 고객경험 6단계(일정 준비 -> 일정 실행 -> 돌발 상황 인지 -> 대안 탐색 -> 대안 선택 및 일정 재조정 -> 변경 일정 실행 및 평가)에서 여행자가 실제로 겪는 경험을 체험자 관점에서 기록하고, 서비스 개선 기회를 도출한다.
- **조사 방법**: 조사원이 각 세그먼트(솔로, 커플/친구, 초보, 가족, 리피터)의 입장에서 직접 여행자 역할을 수행하며 돌발 상황 대응 과정을 체험하고 기록
- **조사 기간**: 2026년 1월 ~ 2월
- **대상 여행지**: 도쿄, 오사카, 방콕, 파리, 런던, 바르셀로나, 싱가포르 등 해외 주요 여행지
- **고객경험 단계**: 일정 준비 -> 일정 실행 -> 돌발 상황 인지 -> 대안 탐색 -> 대안 선택 및 일정 재조정 -> 변경 일정 실행 및 평가

## 체험 1: 도쿄 솔로 트래블러 -- 갑작스러운 폭우 대응

### 체험 프로필
| 항목 | 내용 |
|------|------|
| 세그먼트 역할 | S1. 효율 추구형 솔로 트래블러 |
| 여행지 | 도쿄 (시부야, 하라주쿠, 오모테산도) |
| 여행 일차 | 3박 4일 중 2일차 |
| 동행 구성 | 혼자 |
| 체험자 설정 | 28세 직장인, 연차를 내고 3박 4일 도쿄 여행 중, 도쿄 2회차 방문 |

### 체험 시나리오
- **원래 계획**: 오전 메이지신궁 산책(10:00) -> 하라주쿠 다케시타도리 쇼핑(11:30) -> 오모테산도 카페 런치(13:00) -> 시부야 스크램블 교차로 사진 촬영(14:30) -> 시부야 스카이 전망대(16:00) -> 에비스 맥주 박물관(18:00)
- **돌발 상황**: 오전 11시경 메이지신궁 산책 중 갑자기 강한 폭우가 시작됨. 일기예보에는 "흐림, 오후 약간의 비" 정도였으나 실제로는 오전부터 시간당 20mm의 폭우. 우산도 없고, 이후 계획한 실외 일정 대부분이 진행 불가능한 상황

### 고객경험 단계별 경험 내용

#### 1단계: 일정 준비
- 전날 밤 숙소에서 구글맵으로 동선을 확인하고, 노션에 시간대별 일정을 정리해둠
- 날씨 앱(AccuWeather)에서 "흐림, 오후 30% 비 확률"을 확인하고 "접이식 우산 정도면 되겠다" 판단
- 메이지신궁 영업시간(일출~일몰)을 구글에서 검색하여 확인
- 하라주쿠~시부야 도보 동선을 구글맵에 저장해둠
- 소요 시간: 약 40분

#### 2단계: 일정 실행
- 아침 9시 30분 숙소(신주쿠) 출발, JR야마노테선으로 하라주쿠역 도착
- 메이지신궁 도리이를 지나 본전까지 산책 시작. 숲길이 쾌적하고 기분이 좋았음
- 10시 40분경 본전 참배 후 숲길을 되돌아오는 중, 하늘이 급격히 어두워짐
- 일정 앱(트리플)과 구글맵을 번갈아 보며 다음 일정 이동 시간을 체크하고 있었음

#### 3단계: 돌발 상황 인지
- 11시경 갑자기 강한 빗줄기가 쏟아짐. 나무 아래로 피했지만 옷이 금세 젖기 시작
- 날씨 앱을 다시 확인하니 "현재 강수량 시간당 20mm, 오후 3시까지 지속 예상"으로 예보가 변경되어 있었음
- "이후 일정이 전부 실외인데 어떡하지?" 하는 당황감이 밀려옴
- 메이지신궁 입구 매점에서 비닐 우산(500엔)을 구매했지만, 폭우 수준이라 우산만으로는 돌아다니기 어려웠음
- 혼자이기 때문에 의논할 사람이 없었고, "빨리 대안을 찾아야 시간 낭비를 줄일 수 있다"는 초조함이 컸음

#### 4단계: 대안 탐색
- 하라주쿠역 근처 카페(스타벅스)에 들어가 비를 피하며 스마트폰으로 대안 검색 시작
- 구글맵에서 "하라주쿠 실내 관광"을 검색했으나 영어로 된 결과가 대부분이고, 관광지인지 상업시설인지 구분이 어려웠음
- 네이버 블로그에서 "도쿄 비올 때 갈 곳"을 검색하니 2년 전 포스팅이 대부분. 추천된 장소 중 일부는 이미 폐업한 것으로 보였음
- 인스타그램에서 "#도쿄비오는날" 해시태그를 검색하여 몇 가지 힌트를 얻었으나, 위치와 영업시간을 별도로 확인해야 했음
- 대안 후보: (1) 오모테산도힐즈 실내 쇼핑, (2) 네즈미술관, (3) 시부야 109 쇼핑, (4) 도쿄도미술관(우에노)
- 각 장소의 영업시간, 현재 위치에서의 거리, 리뷰를 하나하나 비교하는 데 약 35분 소요
- "이 시간에 여행을 하고 있어야 하는데 카페에서 검색만 하고 있다"는 좌절감

#### 5단계: 대안 선택 및 일정 재조정
- 최종 선택: 도보 5분 거리의 오모테산도힐즈(실내 쇼핑/카페) -> 네즈미술관(13:30) -> 시부야 스카이 전망대는 유지(비가 와도 실내 전망대이므로)(16:00) -> 에비스 맥주 박물관 유지(18:00)
- 하라주쿠 다케시타도리 쇼핑과 시부야 스크램블 교차로 사진 촬영은 내일로 연기 결정
- 구글맵에서 이동 경로를 다시 검색하고, 각 장소의 소요 시간을 머릿속으로 재계산
- 노션 일정표를 수동으로 수정하는 데 추가 10분 소요
- "내일 날씨가 맑으면 오늘 못 간 곳을 가면 되니까" 하고 자기 위안

#### 6단계: 변경 일정 실행 및 평가
- 오모테산도힐즈에서 예상외로 좋은 디자인 소품샵을 발견하여 만족스러운 쇼핑
- 네즈미술관은 비 오는 날 정원의 분위기가 독특하여 "오히려 비 와서 더 좋았다"는 느낌
- 시부야 스카이 전망대에서 비 내리는 도쿄 야경을 감상. 기대와 다른 풍경이었지만 나름 운치가 있었음
- 하루를 돌아보니, 검색에 허비한 35분이 아까웠고, "누군가 바로 대안을 알려줬으면 1시간은 더 여행할 수 있었다"는 생각
- 내일 일정에 오늘 못 간 곳을 추가해야 하는데, 이미 내일 일정도 빠듯하여 다시 조정이 필요

## 체험 2: 오사카 커플 여행 -- 인기 라멘집 2시간 대기

### 체험 프로필
| 항목 | 내용 |
|------|------|
| 세그먼트 역할 | S2. 유연한 계획형 커플/친구 여행자 |
| 여행지 | 오사카 (도톤보리, 신사이바시, 난바) |
| 여행 일차 | 4박 5일 중 3일차 |
| 동행 구성 | 커플 2인 |
| 체험자 설정 | 29세 여성 직장인 + 31세 남성 직장인 커플, 일본 2회차 |

### 체험 시나리오
- **원래 계획**: 오전 구로몬시장 브런치(10:00) -> 신사이바시 쇼핑(12:00) -> 도톤보리 이치란라멘 점심(14:00) -> 난바 워크 쇼핑(15:30) -> 신세카이 통천각(17:00) -> 잔니비 야키니쿠 저녁(19:00)
- **돌발 상황**: 도톤보리 이치란라멘 본점에 14시에 도착했으나 대기 줄이 약 100명. 예상 대기시간 2시간. 오후 일정 전체가 밀릴 상황. 커플 간 "기다릴까 vs 다른 데 갈까" 의견 충돌 발생

### 고객경험 단계별 경험 내용

#### 1단계: 일정 준비
- 전날 숙소에서 카카오톡으로 내일 일정을 서로 공유. "대충 이렇게 가자" 수준의 느슨한 계획
- 남자친구는 이치란라멘을 꼭 가고 싶어하고, 여자친구는 신사이바시 쇼핑을 우선시
- 트리플 앱에서 각 장소를 저장해두었으나 시간 배분은 하지 않음
- 구로몬시장 영업시간(대부분 9시~17시)만 대략 확인
- 소요 시간: 약 20분 (대화 포함)

#### 2단계: 일정 실행
- 구로몬시장에서 참치회와 딸기 모찌를 먹으며 브런치. 시장 구경이 재미있어 예정보다 40분 더 머무름
- 신사이바시 쇼핑은 여자친구 위주로 진행. 남자친구는 조금 지루해하며 "라멘 먹으러 가자" 독촉
- 쇼핑이 길어져 이치란라멘 도착이 원래 14시 계획에서 14시 20분으로 밀림
- 이동 중 커플 간 약간의 감정 마찰 ("쇼핑 너무 오래 한 거 아니야?" vs "시장에서 네가 오래 있었잖아")

#### 3단계: 돌발 상황 인지
- 이치란라멘 본점 앞에 도착하니 가게 밖으로 길게 줄이 늘어서 있음. 스태프에게 물어보니 "약 2시간 대기"
- 남자친구: "여기까지 왔는데 기다리자." 여자친구: "2시간이나? 다른 라멘집 가면 안 돼?"
- 이미 점심시간을 한참 넘기고 있어 배가 고픈 상태에서 의견 대립
- "여기서 2시간을 쓰면 통천각이랑 저녁 야키니쿠 시간이 다 밀리는데..."라는 계산이 머릿속에서 시작
- 분위기가 약간 냉랭해짐

#### 4단계: 대안 탐색
- 여자친구가 구글맵에서 "도톤보리 라멘" 검색. 결과가 수십 개 나오지만 리뷰 수와 평점이 제각각
- 남자친구는 네이버에서 "오사카 이치란 대신 라멘" 검색. 블로그 추천이 여러 개 나오지만 위치가 멀거나 정보가 오래됨
- 두 사람이 각자 다른 장소를 추천하며 "여기가 낫지 않아?" "아니 이 블로그 보니까 저기가 더 맛있대" 논쟁
- 대안 후보: (1) 카무쿠라 라멘 (도보 3분), (2) 킨류라멘 (도보 1분, 저렴), (3) 잇푸도 라멘 난바점 (지하철 1정거장), (4) 이치란 대기 유지
- 각자의 폰으로 따로 검색하고 있어 정보가 공유되지 않고, "내가 찾은 게 더 나은데" 식의 비효율적 논쟁 지속
- 대안 탐색에 약 25분 소요. 배고픔과 짜증이 겹쳐 분위기 악화

#### 5단계: 대안 선택 및 일정 재조정
- 결국 여자친구의 제안으로 도보 3분 거리 카무쿠라 라멘으로 결정. 남자친구는 약간 불만이지만 동의
- 오후 일정을 머릿속으로 재계산: 카무쿠라 라멘(14:50~15:30) -> 난바 워크 축소(15:40~16:30) -> 통천각(17:00) -> 야키니쿠(19:00)는 유지

## 체험 3: 방콕 초보 자유여행자 -- 교통 파업과 사원 임시 폐쇄

### 체험 프로필
| 항목 | 내용 |

| 여행지 | 방콕 (왕궁, 왓포, 카오산로드) |
| 여행 일차 | 3박 5일 중 2일차 |
| 동행 구성 | 혼자 (첫 해외 자유여행) |
| 체험자 설정 | 24세 대학생, 패키지 여행 2회 경험만 있고 자유여행은 처음. 태국어 전혀 못하고 영어도 기초 수준 |

### 체험 시나리오
- **원래 계획**: 오전 왕궁(Grand Palace) 관람(09:00) -> 왓포(Wat Pho) 방문(11:00) -> 카오산로드 점심 및 산책(13:00) -> 짜뚜짝 주말시장(15:00) -> 아시아티크 야시장(18:00)
- **돌발 상황**: (1) 숙소에서 왕궁까지 BTS+보트를 이용하려 했으나 BTS 시스템 점검으로 일부 구간 운행 중단. (2) 왕궁에 겨우 도착했으나 "국가 행사로 인한 임시 폐쇄" 안내. 태국어 안내문만 있어 이유를 정확히 파악하지 못함

### 고객경험 단계별 경험 내용

#### 1단계: 일정 준비
- 전날 밤 네이버 블로그에서 "방콕 2일차 코스"를 검색하여 추천 코스를 거의 그대로 따라감
- 왕궁 입장료(500바트), 드레스코드(긴팔·긴바지), 운영시간(08:30~15:30)을 블로그에서 확인
- BTS 노선도를 캡처해두고, 사피안탁신역에서 보트 환승하는 방법을 메모
- 일정을 30분 단위로 촘촘하게 짜둠. "이대로만 하면 완벽한 하루"라고 생각
- 소요 시간: 약 1시간 (꼼꼼하게 블로그 확인)

#### 2단계: 일정 실행
- 아침 8시 숙소(수쿰빗) 출발. BTS역으로 걸어가 개찰구에서 래빗카드를 터치
- 그런데 전광판에 태국어로 안내가 나오고, 역 직원이 손짓으로 "이 구간 안 됨"을 표현
- 영어로 물어봐도 직원이 영어를 잘 못해서 소통이 되지 않음
- 주변 승객에게 물어보니 한 사람이 영어로 "시스템 점검, 오후에 복구될 수도 있다"고 알려줌
- 이 시점에서 이미 심리적 패닉 시작. "어떻게 왕궁까지 가지?"

#### 3단계: 돌발 상황 인지
- BTS 대안으로 택시를 탔으나, 방콕의 심한 교통 체증으로 평소 30분 거리가 1시간 15분 소요
- 10시 15분에 왕궁에 도착했으나 정문에 태국어 안내판과 함께 출입이 통제되어 있음
- 주변 관광객에게 물어보니 "국가 행사로 오늘 문 닫았다"는 정보를 얻음
- 구글에서 "Grand Palace Bangkok closed today"를 검색했으나, 실시간 정보가 나오지 않음. 공식 홈페이지는 태국어로만 안내
- 두 번의 돌발 상황이 연속으로 발생하여 극심한 당황과 불안. "첫 자유여행인데 벌써 다 틀어졌다"
- 택시비 350바트(약 1만 3천원)를 헛되이 쓴 것에 대한 허탈감
- "자유여행은 역시 나한테 안 맞나 봐"라는 자기 의심까지 발생

#### 4단계: 대안 탐색
- 왕궁 앞 편의점에서 물을 사며 에어컨 바람을 쐬면서 검색 시작
- 네이버에서 "방콕 왕궁 대신 갈 곳"을 검색. 왓아룬, 왓포가 근처에 있다는 정보 확인
- 왓포가 왕궁에서 도보 10분 거리이고 별도 운영이라 열려 있을 가능성이 있다고 판단
- 그러나 "왓포도 혹시 닫았으면 어쩌지?"라는 불안으로 전화를 걸어보려 했으나, 전화번호를 찾기 어렵고 태국어 전화가 불가
- 구글맵 리뷰에서 "오늘 다녀왔어요" 류의 최신 리뷰를 찾아보며 영업 여부를 간접 확인하려 시도
- 짜뚜짝 시장은 주말에만 열리는데, 오늘이 평일임을 뒤늦게 깨달음 (블로그에서 "주말시장"이라고 적혀 있었는데 간과함)
- 대안 탐색에 약 30분 소요. 정보의 신뢰성에 대한 불안이 계속됨

#### 5단계: 대안 선택 및 일정 재조정
- 최종 결정: 왓포(도보 이동, 10:50) -> 왓아룬(보트로 이동, 12:30) -> 카오산로드 점심(14:00) -> 짜뚜짝 대신 터미널21 쇼핑몰(16:00, BTS 복구 기대) -> 아시아티크(18:30)
- 그런데 이동 경로를 재계산하니, 왓아룬에서 카오산로드까지의 이동 수단이 불분명
- 택시? 보트? 뚝뚝? 각각의 요금과 소요 시간을 구글맵에서 확인했으나, 방콕 택시의 미터기 사용 여부 등이 불안
- 일정표를 스마트폰 메모에 다시 작성. 시간 계산이 맞는지 확인하는 데 추가 15분
- "BTS가 오후에 복구되는지 안 되는지 모르겠는데, 복구 안 되면 터미널21도 못 가는 거 아닌가?"

#### 6단계: 변경 일정 실행 및 평가
- 왓포에 도착하니 정상 운영 중. 와불(거대 누운 부처)을 보고 감탄. 왕궁보다 오히려 더 인상적이라는 생각
- 왓아룬으로 가는 나룻배(4바트)를 타는 경험이 재미있었음
- 카오산로드에서 팟타이를 먹었으나, 관광객 대상 가격(일반 가격의 2배)이라는 것을 나중에 알게 됨
- BTS는 오후 3시에 복구되어 터미널21로 이동 성공. 아시아티크도 예정대로 방문
- 숙소 복귀 후 느낀 점: "결과적으로 나쁘지 않았지만, 아침에 2시간 동안 멘탈이 흔들렸던 게 체력적으로 많이 소모됨"
- "교통 상황이나 관광지 폐쇄를 미리 알 수 있었다면 아침 시간을 허비하지 않았을 텐데"
- "짜뚜짝이 평일에는 안 한다는 것도 미리 알려줬으면 좋겠다"

### 체험 시나리오

- **돌발 상황**: (1) 사그라다 파밀리아 관람 후 산 파우 병원으로 이동하려는데 지하철 2호선/4호선 부분 파업으로 운행 불규칙. (2) 고딕지구 골목에서 소매치기 시도를 당함(미수). 심리적 충격으로 여행 의욕 급락

### 고객경험 단계별 경험 내용

#### 1단계: 일정 준비
- 사그라다 파밀리아 입장권을 2주 전 온라인 예약. 타워 입장 포함 36유로
- 구글맵에서 각 장소 간 이동 경로를 지하철 기준으로 확인하고 스크린샷 저장
- 바르셀로나 교통카드(T-Casual, 10회권 11.35유로) 구매 계획
- 날씨는 맑음, 25도로 쾌적. 특별한 변수는 예상하지 않음
- 고딕지구에서 점심 먹을 레스토랑 후보 3곳을 네이버 블로그에서 확인
- 소요 시간: 약 35분

#### 2단계: 일정 실행
- 사그라다 파밀리아 09:30 입장. 내부와 타워 모두 압도적이어서 예상(1.5시간)보다 30분 더 머무름
- 11시 사그라다 파밀리아 앞 지하철 L2역에서 산 파우 병원으로 이동하려는데, 역 입구에 스페인어 안내문이 붙어 있음
- 구글 번역으로 촬영하니 "부분 파업 안내: L2, L4 노선 12시~16시 사이 간헐적 운행"
- "오후 일정 전체가 지하철 이동인데..." 하는 걱정이 시작됨

#### 3단계: 돌발 상황 인지
- 역 안에서 20분을 기다렸으나 열차가 오지 않아 지상으로 나옴
- 구글맵에서 산 파우 병원까지 도보 15분인 것을 확인하고 걸어서 이동
- 산 파우 병원 관람 후 고딕지구로 이동해야 하는데, 지하철을 쓸 수 없는 상황
- 고딕지구까지 버스 또는 택시를 이용해야 하나, 버스 노선을 모르고 택시가 잘 안 잡힘
- 결국 버스(V15)를 구글맵 경로 안내로 타고 고딕지구에 도착. 예상보다 40분 더 소요
- 고딕지구 골목을 걷다가, 좁은 골목에서 누군가 가방 지퍼를 열려는 것을 느낌. 재빨리 몸을 돌려 소매치기 미수에 그침
- 물건은 잃지 않았으나, 심리적 충격이 큼. "혼자인데 이런 일을 당하면 어떡하지" 불안감 급증
- 여행 의욕이 급격히 떨어지고, "안전한 곳으로 가고 싶다"는 생각이 지배적

- "바르셀로나 안전한 동네"를 검색하여 에이샴플라(Eixample) 지구가 비교적 안전하다는 정보 확인
- 그러나 오후 일정(보케리아 시장, 해변)을 계속할지 숙소로 돌아갈지 고민
- 지하철 파업이 언제 끝나는지 확인하려 바르셀로나 교통공사(TMB) 사이트를 방문했으나, 스페인어/카탈루냐어로만 안내되어 정확한 정보 파악 어려움

- 보케리아 시장은 고딕지구에서 도보 10분 거리라 도보 이동 가능. 해변은 도보 25분 또는 버스
- 대안 탐색에 약 20분 소요. 소매치기 충격으로 집중력이 저하된 상태

#### 5단계: 대안 선택 및 일정 재조정
- 결정: 고딕지구에서 점심 먹고 -> 보케리아 시장(도보 이동) -> 해변은 포기하고 에이샴플라의 카사 바트요 외관 감상(대안) -> 숙소 근처에서 저녁
- 바르셀로네타 해변을 포기한 이유: 해변 지역도 소매치기가 많다는 정보를 블로그에서 확인. 심리적으로 사람 많은 곳을 피하고 싶음
- 오후 이동은 모두 도보 또는 택시로 전환. 지하철 이용을 포기
- 저녁 일정도 엘 보른 지구(골목 중심)에서 숙소 근처 에이샴플라 레스토랑으로 변경
- 원래 계획의 효율적 동선이 완전히 무너져 아쉬움. 그러나 안전이 우선이라고 자기 설득

- 고딕지구 레스토랑에서 해산물 파에야 점심. 맛있었으나 관광객 가격(1인 22유로)

- 카사 바트요 외관 감상은 아름다웠으나, 내부 입장(사전 예약 필요)을 못 해 아쉬움
- 숙소 근처 레스토랑에서 혼자 저녁. 솔로 여행의 외로움이 소매치기 사건 이후 더 크게 느껴짐

### 체험 시나리오
- **원래 계획**: 오전 텐노지 동물원(10:00) -> 점심 신세카이 쿠시카츠(12:30) -> 아베노하루카스 전망대(14:30) -> 나라 공원 사슴 먹이주기(16:00, 전철 이동) -> 숙소 복귀(18:30)
- **돌발 상황**: (1) 텐노지 동물원에서 4세 아들이 피곤해 보채기 시작하여 예정보다 1시간 일찍 나옴. (2) 신세카이 쿠시카츠집에 입장했으나 메뉴에 알레르기 정보가 일본어로만 표기되어 있고, 직원과 소통 불가. 갑각류(새우) 포함 여부를 확인할 수 없어 식사를 포기하고 나옴

### 고객경험 단계별 경험 내용

#### 1단계: 일정 준비
- 전날 숙소에서 부부 회의. 아이들 체력을 고려해 오전/오후 각 1개 메인 일정 + 여유 시간 배치
- 텐노지 동물원 입장료(어른 500엔, 초등 이하 무료) 확인
- 신세카이 쿠시카츠집은 네이버 블로그 추천 가게 2곳을 리스트업
- 아들의 갑각류 알레르기를 고려하여 "새우가 들어가지 않는 메뉴를 골라야 한다"고 메모
- 일본어로 "새우 알레르기가 있습니다(エビアレルギーがあります)" 문구를 번역 앱에서 준비
- 나라 공원까지의 전철 경로(JR+킨테츠, 약 50분)를 확인
- 소요 시간: 약 40분

#### 2단계: 일정 실행
- 텐노지 동물원 10시 입장. 7세 딸은 동물들을 보며 즐거워했으나 4세 아들은 걷는 것을 힘들어함
- 유모차를 가져왔으나 동물원 내 경사진 구간이 많아 유모차 이동이 불편
- 11시경 아들이 "안아줘" "가기 싫어"를 반복하며 보챔. 울기 직전 상태
- 엄마가 아들을 안고, 아빠가 유모차를 밀고, 딸은 "나도 안아줘" 상태
- 예정(12시)보다 1시간 일찍 11시에 동물원에서 나오기로 결정

#### 3단계: 돌발 상황 인지
- 동물원을 일찍 나왔으나 점심 예정 시간(12:30)까지 1시간 30분이 비는 상황
- "차라리 점심을 앞당기자"고 결정하여 신세카이 쿠시카츠 거리로 이동 (도보 15분)

- 준비해둔 "エビアレルギーがあります" 문구를 직원에게 보여줬으나, 직원이 고개를 갸웃하며 주방에 물어보러 감
- 주방에서 돌아온 직원이 일본어로 길게 설명하는데, 부부 모두 일본어를 이해하지 못함
- 구글 번역 카메라로 메뉴를 촬영해보니 "えび(새우)" 글자가 여러 메뉴에 포함되어 있는 것 같지만, 번역 품질이 낮아 확신 불가

- "키즈 메뉴가 있고, 알레르기 정보가 명확하고, 지금 자리가 있는 식당"을 한 번에 찾고 싶지만 그런 정보를 통합 제공하는 곳이 없음

- 오후 일정 재검토: 아베노하루카스(13:30으로 앞당김) -> 나라 공원은 이동 시간(편도 50분)과 아들 체력을 고려하여 포기

- 엄마: "나라 공원 사슴 보여주고 싶었는데..." 딸: "사슴 보고 싶었어..." 아쉬움

- 사이제리야에서 아이들이 잘 먹고 기분이 좋아짐. 가격도 저렴(4인 3,200엔)

#### 1단계: 일정 준비
- 벚꽃 개화 예보 사이트(tenki.jp)에서 도쿄 만개일을 확인. "3월 29일 만개 예상"이라 4월 1일 방문을 계획
- 인스타그램에서 "#나카메구로벚꽃" "#目黒川桜" 검색하여 사진 포인트 확인

#### 2단계: 일정 실행

#### 3단계: 돌발 상황 인지
- 나카메구로에서 20분 만에 철수 결정. "이건 벚꽃 감상이 아니라 인파 체험이다"

- 벚꽃 상태도 기대 이하여서 이중으로 실망. "개화 예보를 믿었는데 이미 져가고 있다"

- 지도리가후치 근처 카페에서 검색 시작

## 2. 가장 만족도가 낮은 단계와 이유

### 최저 만족도: 3단계 "돌발 상황 인지" (평균 1.6)

**이유 분석:**

1. **심리적 충격의 즉시성**: 돌발 상황은 예고 없이 발생하며, "계획이 틀어졌다"는 사실 자체가 강한 스트레스를 유발한다. 특히 초보 여행자(체험3: 1.0, 체험8: 1.5)와 가족 여행자(체험4: 1.5, 체험9: 1.0)에서 만족도가 극도로 낮았다.

2. **정보 부재에 의한 불안 증폭**: 돌발 상황의 원인, 지속 시간, 영향 범위를 알 수 없어 불안이 가중된다. 교통 파업이 언제 끝나는지(체험3, 체험6), 비가 얼마나 올지(체험1, 체험7), 관광지가 언제 다시 여는지(체험3) 등의 정보가 부재하다.

3. **혼자 감당해야 하는 부담**: 솔로 여행자(체험1, 체험3, 체험6)는 의논할 상대 없이 혼자 판단해야 하는 부담이 크고, 그룹 여행(체험2, 체험7)에서는 의견 충돌이 스트레스를 더한다.

4. **연쇄 영향에 대한 걱정**: "이 일정이 틀어지면 나머지도 다 엉키는 거 아닌가?"라는 불안이 10건 중 8건에서 관찰되었다.

### 차저 만족도: 4단계 "대안 탐색" (평균 1.9)

**이유 분석:**

1. **정보 파편화**: 대안을 찾기 위해 구글맵, 네이버, 인스타그램, 트립어드바이저 등 평균 3~4개 앱을 번갈아 사용한다. 통합된 정보 소스가 없어 탐색 시간이 평균 24분으로 길다.

2. **정보 신뢰도 불안**: 블로그 후기가 오래되었거나(체험8), 영업시간이 변경되었을 수 있어(체험5), 찾은 정보를 믿을 수 없다.

3. **언어 장벽**: 현지어로 된 정보에 접근하기 어렵고(체험3, 체험9), 구글 번역의 정확도가 낮다.

4. **탐색 자체가 시간 낭비**: 대안을 찾는 동안 여행 시간이 줄어든다는 역설. 10건 평균 24분을 대안 탐색에 소모했다.

## 5. 핵심 개선 기회 5가지

### 기회 1: 선제적 돌발 상황 알림 시스템
- **근거**: 10건 중 9건에서 현장 도착 후 돌발 상황을 인지. 사전에 알았다면 시간과 에너지를 절약할 수 있었음
- **내용**: 날씨 급변(30분 전 알림), 교통 장애(실시간 모니터링), 관광지 임시 폐쇄(당일 오픈 여부 확인), 혼잡도 경고(피크 시간 회피 추천), 정기 휴일 경고(일정 등록 시 자동 체크)
- **기대 효과**: 돌발 상황 인지 단계(현재 1.6)의 만족도를 3.0 이상으로 끌어올릴 수 있음. "알았으면 안 갔다"는 후회를 제거
- **관련 체험**: 체험1(폭우), 체험3(교통 파업+폐쇄), 체험5(정기 휴일), 체험6(파업), 체험8(예약제 변경), 체험10(혼잡도)

### 기회 2: AI 기반 즉시 대안 추천 (3분 이내)
- **근거**: 대안 탐색에 평균 23.5분 소요. 3~4개 앱을 번갈아 사용하는 정보 파편화가 핵심 원인
- **내용**: 현재 위치, 시간, 날씨, 동행 구성, 선호도를 종합하여 3분 이내에 검증된 대안 3개를 제시. 영업 여부 실시간 확인, 이동 시간/경로 포함, 리뷰 요약 제공
- **기대 효과**: 대안 탐색 시간을 23.5분에서 3분으로 단축. 대안 탐색 단계(현재 1.9) 만족도를 3.5 이상으로 향상
- **관련 체험**: 전체 10건

### 기회 3: 일정 자동 재조정 엔진
- **근거**: 8건에서 일정 수동 재조정의 번거로움이 Pain Point로 지적됨. "한 곳이 바뀌면 나머지를 머릿속으로 재계산"
- **내용**: 대안 장소 선택 시 나머지 일정의 순서, 시간 배분, 동선, 교통수단을 자동으로 재최적화. "이 일정을 포기하면 / 내일로 미루면" 시나리오별 비교 제공
- **기대 효과**: 대안 선택 및 일정 재조정 단계(현재 2.5) 만족도를 3.5 이상으로 향상. 의사결정 시간 단축 및 확신 증가
- **관련 체험**: 체험1, 2, 3, 4, 6, 7, 8, 9

### 기회 4: 세그먼트 맞춤 정보 필터
- **근거**: 세그먼트별로 필요한 정보가 크게 다르나, 기존 앱은 범용 정보만 제공
- **내용**:

  - **초보**: 한국어 단계별 안내, 이동수단 비교, 안심 알림

  - **리피터**: 현지인 추천, 실시간 혼잡도, 숨은 명소, 시간대별 맞춤
- **기대 효과**: 전체 만족도(현재 2.6)를 세그먼트별로 3.5 이상으로 향상. 개인화된 경험 제공
- **관련 체험**: 전체 10건

### 기회 5: 실시간 현지 상태 검증 시스템
- **근거**: 7건에서 언어 장벽, 9건에서 실시간 정보 부재가 지적됨. "가봤더니 문 닫았다", "블로그 정보가 오래됐다"

- **기대 효과**: "가봤더니 문 닫았다" 제로화, 블로그 정보 의존도 감소, 언어 장벽으로 인한 정보 접근 제한 해소
- **관련 체험**: 체험3, 4, 5, 6, 8, 9, 10

## 종합 결론

10회의 체험 조사를 통해 확인된 핵심 사실은 다음과 같다.

1. **돌발 상황 자체보다 "대응 과정"이 더 큰 고통**: 여행 중 돌발 상황은 불가피하다. 그러나 대응 과정(대안 탐색, 일정 재조정)에서 평균 23.5분을 허비하고, 정보 파편화, 언어 장벽, 의견 충돌로 스트레스가 가중되는 것이 진짜 문제이다.

2. **V자 감정 곡선의 바닥이 서비스 기회**: 만족도가 가장 낮은 "돌발 상황 인지(1.6)" ~ "대안 탐색(1.9)" 구간에 서비스가 집중 개입하면, 이 바닥을 얕게 만들어 전체 여행 만족도를 크게 향상시킬 수 있다.

3. **초보 여행자가 가장 취약**: 전체 만족도 평균 2.3으로, 경험 부족 + 언어 장벽 + 심리적 불안이 겹쳐 돌발 상황에 가장 취약하다. 그러나 이들이 서비스를 가장 절실히 필요로 하는 세그먼트이기도 하다.

4. **"대안이 오히려 더 좋았다" 경험의 반복**: 10건 중 7건에서 대안 장소가 원래 계획 못지않게, 또는 더 좋은 경험을 제공했다. 이는 "돌발 상황 = 나쁜 것"이 아니라 "더 좋은 발견의 기회"로 전환할 수 있음을 보여준다. 서비스는 이 전환을 빠르고 매끄럽게 만들어주는 것이 핵심 가치이다.

5. **MVP 최우선 기능**: 선제적 알림, 3분 이내 대안 추천, 일정 자동 재조정. 이 세 가지가 전 세그먼트에 걸쳐 가장 높은 빈도로 요청된 기능이며, "대안 탐색 23.5분 -> 3분" 단축이 서비스의 핵심 가치 제안(Value Proposition)이 된다.
