## 6. v1 대비 개선 사항 (리디자인)

| 항목 | v1 (prototype-backup) | v2 (prototype) |
|------|----------------------|----------------|
| CSS 클래스 네이밍 | 단순 클래스명 | BEM 유사 구조화 네이밍 |
| 헤더 스타일 | 기본 sticky | sticky + border-bottom 구분선 |
| 바텀시트 | 기본 다이얼로그 | dialog role + 핸들바 개선 |
| 장소 카드 | 기본 카드 | aria-label 강화, 상태 설명 포함 |
| 검색 UX | 즉시 결과 | 로딩 시뮬레이션 + 스켈레톤 UI |
| 장소 상세 | 바로 시간지정 이동 | 바텀시트 상세 -> 일정에 추가 2단계 |
| 접근성 | 기본 ARIA | ARIA 레이블 수 2~3배 증가 |
