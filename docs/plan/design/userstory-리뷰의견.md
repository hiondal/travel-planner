## 2. 개선 의견 (우선순위별 정리)

| # | 개선 항목 | 대상 UFR/NFR | 제안자 | 내용 요약 |

| U3 | **MoSCoW 승격: 대안 미선택 유지** | UFR-ALTN-040 | 피오 | Should→Must 승격. S06 UX 완결성 + 데이터 플라이휠 초기 씨앗. 3SP로 공수 미미 |
| U4 | **외부 API 병렬 호출 전략 명시** | UFR-MNTR-010 | 아키 | 3개 API 순차 호출 시 3~4.5초 소요 → NFR-PERF-010(p95 2초) 위반. 병렬 호출 + 타임아웃 2초 체크리스트 추가 |
| U5 | **상태 판정 임계값 정의** | UFR-MNTR-020 | 아키 | "양호", "보통 이하", "정상"의 정량적 임계값 부재. 설정 파일 관리 + MVP 초기 임계값 명시 |

| U10 | **만료 브리핑 유효성 처리** | UFR-BRIF-050 | 인사이트 | 오프라인 후 지연 수신 시 출발 시간 이후 브리핑 무의미. "만료 브리핑" 표시 + 최신 상태 조회 버튼 |

## 8. AI 고도화 로드맵 (마법사)

```
MVP (Rule-based)          Phase 2 (데이터 기반)     Phase 3 (AI 기반)
─────────────────────     ─────────────────────     ─────────────────────
상태 판정: 고정 임계값  →  임계값 통계 최적화      →  ML 분류 모델
대안 정렬: 거리>평점>   →  가중치 A/B 테스트      →  Learning-to-Rank
          혼잡도 고정                                  (개인화)
브리핑 총평: 템플릿     →  템플릿 A/B 테스트       →  LLM 자연어 생성

개인화: 없음            →  스타일 클러스터링       →  사용자별 추천
```
