## Needs Statement

> **"해외 자유여행 중인 효율 추구형 솔로 트래블러는, 예상치 못한 상황 변화로 인한 감정 절벽과 시간 매몰을 사전에 차단하기 위하여, 목적지에 도착하기 전에 여행 흐름에 영향을 줄 수 있는 변화를 미리 알 수 있는 경험이 필요하다."**

---

## 상세 설명

### 고객유형

**해외 자유여행 중인 효율 추구형 솔로 트래블러 (25~34세)**

- 연 1~3회 해외여행, 주로 3~5일 단기 자유여행을 하는 직장인 또는 프리랜서.
- "시간 낭비"를 가장 싫어하며, 시간을 돈으로 환산하는 성향 (S1: "30분 = 돈").
- 혼자이기 때문에 의사결정이 빠르지만, 잘못된 선택의 리스크도 혼자 감당한다.
- 부정적 감정을 전환해줄 동행이 없어, 사후 인지 시 "감정 고립"이 가장 심각한 세그먼트.
- 빡빡한 일정을 짜는 경향이 있어 연쇄 붕괴 시 도미노 효과가 가장 크다.

이 세그먼트를 최우선으로 선정한 이유:
1. 인터뷰 검증에서 B(사후 인지)의 감정적 고통이 가장 심각 -- 동행 부재로 감정 고립
2. 의사결정이 빠르므로 사전 정보만 제공되면 즉시 행동으로 전환 -- 서비스 가치를 가장 빠르게 체감
3. WTP 5,000~10,000원/여행건 확인 -- 유료 전환 가능성 검증 완료
4. 한국 MZ세대의 29%가 1인 여행 경험 보유 -- 시장 규모 충분

### 목적

**예상치 못한 상황 변화로 인한 감정 절벽과 시간 매몰을 사전에 차단하기 위하여**

이 목적은 킹핀 문제(B. 사후 인지)의 핵심 근본원인과 직접 연결된다.

| 킹핀 문제의 근본원인 | 목적과의 연결 |
|-------------------|------------|
| 여행 도구가 Pull 모델 -- 사용자가 직접 찾아야 정보 제공 | "사전에 차단" -- Pull이 아닌 선제적 방식으로의 전환 방향 |
| 도착 후에야 인지 -- 이동 시간 15~75분 매몰 | "시간 매몰을 차단" -- 이동 전 인지로 시간 손실 방지 |
| 감정 곡선 3.6->1.6 급락 | "감정 절벽을 차단" -- 감정 급락이 발생하지 않는 경험 |

"감정 절벽"을 "시간 매몰"보다 앞에 배치한 이유:
- 인터뷰 3명 전원이 "시간 절약보다 감정 보호가 더 중요하다"고 독립적으로 확인
- 감정 보호가 리텐션과 유료 전환의 진짜 트리거 (피보팅 핵심 인사이트)
- 시간 매몰은 감정 절벽의 물리적 원인이자 결과이므로, 감정 절벽 차단이 상위 목적

### 원하는 것

**목적지에 도착하기 전에 여행 흐름에 영향을 줄 수 있는 변화를 미리 알 수 있는 경험**

이 표현이 의도하는 바:

| 표현 | 의도 | 포함하지 않는 것 |
|------|------|---------------|
| "목적지에 도착하기 전에" | 사후 인지(도착 후)를 사전 인지(도착 전)로 전환하는 시점의 방향성 | 구체적 시점(30분 전, 1시간 전 등)이나 수단(Push 알림, SMS 등) |
| "여행 흐름에 영향을 줄 수 있는 변화" | 임시 휴업, 혼잡, 교통 중단, 날씨 급변 등 포괄적 상황 변화 | 특정 데이터 소스나 감지 기술 |
| "미리 알 수 있는" | 시스템이 아닌 사용자가 주어 -- 사용자가 인지의 주체가 되는 경험 | 자동화 수준이나 정보 전달 방식 |
| "경험" | 기능이 아닌 사용자가 느끼는 전체적 상태 | 구체적 UI, 기능 명세, 기술 스택 |

"경험"이라는 단어를 선택한 이유:
- 인터뷰에서 드러난 핵심 가치는 "감정 보호"이며, 이는 단일 기능이 아닌 경험의 영역
- 같은 "미리 안다"라도 세그먼트별로 다른 경험이 필요함 (솔로: 즉시 대안 판단, 그룹: 공유와 합의, 초보: 가이드 안내)
- 구체적 솔루션을 열어두어 다양한 구현 방식이 가능하도록 방향성만 제시

## 킹핀 문제와의 연결

### Needs Statement가 킹핀 문제를 어떻게 방향 전환하는가

```
[킹핀 문제: 현재 상태]                    [Needs Statement: 지향 상태]

"도착 후에야 알게 된다"          --->     "도착하기 전에 미리 알 수 있다"
 (사후 인지)                              (사전 인지)

감정 곡선 3.6 -> 1.6 급락       --->     감정 곡선 3.6 유지 (절벽 없음)
 (감정 절벽)                              (경험 연속성)

이동 시간 15~75분 매몰          --->     이동 전 판단 -> 매몰 시간 제로
 (시간 매몰)                              (시간 환원)

Pull 모델 (사용자가 찾아야 함)   --->     사용자가 자연스럽게 알게 되는 경험
 (구조적 공백)                            (공백 해소)
```

### B->A->C 인과 체인에서의 위치

Needs Statement는 인과 체인의 최상위(B)를 겨냥한다.

- **B(킹핀) 해결**: "미리 알 수 있는 경험" -> 사후 인지 자체가 발생하지 않음
- **A 연쇄 완화**: 미리 알면 여유 있는 상태에서 대안 탐색 가능 -> 검색 노동 감소
- **C 연쇄 완화**: 사전 인지 -> 사전 조정 가능 -> 연쇄 붕괴 방지

### 비즈니스 가치와의 정합성

| 확정된 비즈니스 가치 | Needs Statement 연결 |
|------|------|
| 고객 1: 감정 절벽 사전 차단 | "감정 절벽을 사전에 차단하기 위하여" -- 직접 연결 |
| 고객 2: 검색 노동 시간 제거 | "시간 매몰을 사전에 차단" + 미리 알면 검색 자체 불필요 -- 연쇄 연결 |
| 고객 3: 의사결정 확신/신뢰 구축 | "미리 알 수 있는 경험"의 전제 조건으로 정보 정확도 90% 필요 -- 기반 연결 |
| 회사 1: During-trip 카테고리 선점 | "도착하기 전에" = During-trip의 핵심 경험 정의 -- 카테고리 정의 자체 |
| 회사 2: 건당 과금 즉시 수익화 | 감정 보호 + 시간 환원 = 프리미엄 과금 근거 -- 과금 정당화 |
| 회사 3: B->A->C 인과 체인 확장성 | B(킹핀)부터 해결 -> A -> C 순차 확장 -- 제품 로드맵 |
